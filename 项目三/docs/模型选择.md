# 模型选择

## 选模型的原则

数学推理微调不要只看模型大小，要看四件事：

- 是否适合数学/推理。
- 是否适合继续训练
- 训练显存能否承受长 CoT。
- 实验目标是“学训练规律”还是“冲榜效果”。

建议区分三类模型：

| 类型 | 例子 | 适合用途 |
|---|---|---|
| Base 模型 | `Qwen/Qwen3-1.7B-Base`、`Qwen/Qwen3-4B-Base` | 从基础模型训练数学推理能力，适合研究 SFT/RL 的真实增益 |
| Math/Reasoning 基础模型 | `Qwen/Qwen2.5-Math-1.5B`、`Qwen/Qwen2.5-Math-7B` | 数学能力底子更好，适合做数学微调 |
| 已蒸馏推理模型 | `deepseek-ai/DeepSeek-R1-Distill-Qwen-1.5B/7B/14B` | 起点强，适合竞赛复现和继续优化，但不适合证明“从零学会推理” |

## 本地跑的模型建议

本地主要目标是验证流程、调数据、跑小实验，不建议硬上 14B。

| 本地显存 | 推荐模型 | 训练建议 | 说明 |
|---|---|---|---|
| 8GB 以下 | `Qwen/Qwen3-0.6B-Base` | LoRA/QLoRA，小 batch，短上下文 | 只适合流程验证 |
| 8GB - 12GB | `Qwen/Qwen3-1.7B-Base` 或 `Qwen/Qwen2.5-Math-1.5B` | QLoRA SFT，控制 `max_length` | 第一版本地首选 |
| 12GB - 16GB | `DeepSeek-R1-Distill-Qwen-1.5B`、`Qwen3-1.7B-Base` | SFT + DPO 都可试 | 适合做小规模对比实验 |
| 24GB | `Qwen/Qwen3-4B-Base`、`Qwen/Qwen2.5-Math-7B`、`DeepSeek-R1-Distill-Qwen-7B` | QLoRA SFT/DPO，长上下文要谨慎 | 本地比较舒服的上限 |
| 48GB 以上 | `Qwen3-8B-Base`、`Qwen2.5-Math-7B` | LoRA 或部分全参实验 | 可做较完整小规模实验 |

本地第一选择：

> `Qwen/Qwen2.5-Math-1.5B` 或 `Qwen/Qwen3-1.7B-Base`

如果你想看明显的 reasoning 输出：

> `deepseek-ai/DeepSeek-R1-Distill-Qwen-1.5B`

但注意：R1-Distill 本来就已经被蒸馏过，适合“继续优化”，不适合当作纯净 baseline。

## 服务器跑的模型建议

| 服务器资源 | 推荐模型 | 适合阶段 | 说明 |
|---|---|---|---|
| 单卡 24GB | `Qwen3-1.7B-Base`、`Qwen2.5-Math-1.5B`、`Qwen3-4B-Base` | SFT/DPO | 适合第一版实验 |
| 单卡 40GB | `Qwen2.5-Math-7B`、`DeepSeek-R1-Distill-Qwen-7B` | SFT/DPO，小规模 GRPO | 性价比高 |
| 单卡 80GB | `Qwen2.5-Math-7B`、`Qwen3-8B-Base` | SFT/DPO/GRPO | 推荐服务器首选 |
| 4 x 80GB | `DeepSeek-R1-Distill-Qwen-14B`、`Qwen3-14B-Base` | SFT/DPO/GRPO | 可以接近 AIMO2 风格实验 |
| 8 x A800/A100/H100 80GB | `DeepSeek-R1-Distill-Qwen-14B`、`Qwen3-14B/32B` | 高质量复现 | AIMO2 方案就是 14B + 8 卡 A800 级别 |

服务器第一选择：

> `Qwen/Qwen2.5-Math-7B` 或 `deepseek-ai/DeepSeek-R1-Distill-Qwen-7B`

服务器进阶选择：

> `DeepSeek-R1-Distill-Qwen-14B`

## 我建议你实际采用的模型阶梯

### 阶梯 1：本地调通

模型：

```text
Qwen/Qwen2.5-Math-1.5B
```

用途：

- 调数据格式。
- 跑小规模 SFT。
- 跑小规模 DPO。
- 验证评估脚本。

### 阶梯 2：服务器主实验

模型：

```text
Qwen/Qwen2.5-Math-7B
```

用途：

- 正式 SFT。
- 正式 DPO。
- MATH-500 / AIME 风格题评估。

### 阶梯 3：竞赛路线复现

模型：

```text
deepseek-ai/DeepSeek-R1-Distill-Qwen-14B
```

用途：

- 复现 AIMO2 风格 SFT + DPO。
- 研究 DPO 是否能压缩输出长度。
- 做多采样、自一致投票和早停策略。

### 阶梯 4：Open-R1 / GRPO 路线

模型：

```text
Qwen/Qwen3-1.7B-Base
Qwen/Qwen3-4B-Base
Qwen/Qwen2.5-Math-7B
```

用途：

- 用 Open-R1 做 GRPO/RLVR。
- 研究 rule-based reward 对小模型数学能力的影响。

## 最终建议

如果只选一个本地模型：

> `Qwen/Qwen2.5-Math-1.5B`

如果只选一个服务器模型：

> `Qwen/Qwen2.5-Math-7B`

如果只选一个竞赛复现模型：

> `deepseek-ai/DeepSeek-R1-Distill-Qwen-14B`

如果只选一个研究 RLVR 的模型：

> `Qwen/Qwen3-1.7B-Base` 起步，之后升到 `Qwen/Qwen3-4B-Base` 或 `Qwen/Qwen2.5-Math-7B`
